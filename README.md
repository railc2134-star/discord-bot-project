# Discord Bot Project

This is a Discord bot project that provides various commands to interact with users. The bot uses Discord.py library and provides commands such as user_info, randomise, delay, roll, and a mini game. The project uses Discord API and Python as the primary technologies.

## Technologies Used
- Discord.py library
- Python 3.x
- Discord API

## How to Run
1. Install the required libraries by running `pip install discord.py` in your terminal.
2. Replace `your_discord_token_here` with your actual Discord bot token.
3. Run the bot by executing `python disco.py` in your terminal.
